# 한국어 일반 상식 객관식 문제 풀이

## 문제 정의
- 다양한 도메인의 상식에 대한 객관식 문제의 정답을 맞추는 LLM 모델 개발
- LLM 학습, 튜닝, 프롬프트 엔지니어링 등과 LLM 활용 데이터 증강 등의 다양한 경험
